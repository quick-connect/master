This example shows how to call a REST API and handle the response.
Before sending the response HTML back to browser, AddressLookupUpdater is compiled and executed.
Then, the Updater inserts two script tags in head tag (WebExtension.js and AddressLookup.js).
Also it adds a Lookup button to fire lookup JavaScript function.
When user clicks on the Lookup, lookup function is invoked and it calls AddressLookup REST APT and receives a response.
Finally, the address in the response is set to the address form on the screen.

This example uses three different types of updaters.
HtmlUpdater inserts <script> tag in this HTML document to use WebExtension features.
XmlUpdater inserts a custom message into the XML document returned from server.
(Original message is "Hello from XML", but "updated by XmlUpdater" is added)
JsonUpdater inserts a custom message into the JSON from server.
(Original message is "Hello from JSON", but "updated by JsonUpdater" is added)
